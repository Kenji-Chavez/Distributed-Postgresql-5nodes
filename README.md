# Distributed PostgreSQL Database – 5 Node Architecture (Northwind)

Este proyecto implementa una base de datos distribuida en PostgreSQL simulando una arquitectura de 5 nodos utilizando Foreign Data Wrappers (FDW) para la comunicación entre nodos.

## Características

* Arquitectura distribuida multi-nodo
* Consultas SQL complejas
* Triggers y procedimientos almacenados
* Cursores y lógica en PL/pgSQL
* Scripts organizados por tipo
* Documentación técnica incluida

## Tecnologías

* PostgreSQL
* SQL
* PL/pgSQL
* Foreign Data Wrappers (FDW)

## Estructura del Proyecto

scripts/ → scripts SQL organizados


docs/ → documentación técnica


tests/ → consultas de prueba

## Objetivo

Simular un entorno distribuido de base de datos empresarial y analizar su comportamiento en consultas y operaciones.
